# Лицензии шрифтов

В проекте используются только шрифты с бесплатной лицензией для коммерческого
использования и web-встраивания.

| Шрифт | Начертания | Лицензия | Официальный источник | Использование |
|---|---|---|---|---|
| Manrope | 300, 400, 500, 600 | SIL Open Font License 1.1 | https://fonts.google.com/specimen/Manrope | Основной текст, формы и интерфейс |
| Prata | 400 | SIL Open Font License 1.1 | https://fonts.google.com/specimen/Prata | Главные заголовки |

Шрифты подключены через официальный сервис Google Fonts. При локальном
размещении необходимо скачать файлы и текст лицензии из официального источника.
